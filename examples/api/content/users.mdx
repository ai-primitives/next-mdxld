# Create User API

This endpoint creates a new user in the system.

## Request Body

```typescript
{
  name: string;
  email: string;
  role?: 'user' | 'admin';
}
```

## Response

```typescript
{
  id: string;
  name: string;
  email: string;
  role: string;
  createdAt: string;
}
```
