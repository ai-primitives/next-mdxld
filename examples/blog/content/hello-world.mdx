# Welcome to next-mdxld

This is an example blog post using next-mdxld.

## Features

- Automatic component selection based on type
- Layout injection from context
- Full Schema.org support
- Customizable components and layouts

## Getting Started

To get started with next-mdxld, simply create a new MDX file in your content directory and add some frontmatter metadata.
